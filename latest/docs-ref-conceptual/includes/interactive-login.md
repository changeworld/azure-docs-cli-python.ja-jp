1. ログイン コマンドを実行します。

    ```azurecli-interactive
    az login
    ```

   使用するコードは次の手順で示します。

1. Web ブラウザーを使用してページ [https://aka.ms/devicelogin](https://aka.ms/devicelogin) を開き、認証のためのコードを入力します。

    資格情報を使用してログインするように求めるメッセージが表示されます。

1. ログインします。
